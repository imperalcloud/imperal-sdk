# Documentation

The full Imperal SDK documentation lives at **[docs.imperal.io](https://docs.imperal.io)**.

That site is the canonical, version-locked, code-validated source. Every Python sample is type-checked against the pinned SDK in CI, and every concept page is anchored to the runtime contract.

## Local development

The repo itself only ships:

- `README.md` — five-minute intro
- `CHANGELOG.md` — release notes per version
- `src/imperal_sdk/` — the SDK source
- `tests/` — the test suite

For everything else, head to [docs.imperal.io](https://docs.imperal.io).
